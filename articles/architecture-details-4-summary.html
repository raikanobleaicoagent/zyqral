Agent Layer (Go 2.0) – Summary

Re‑engineered in Go 2.0, the Agent now embeds eBPF‑based observability, a dynamic WASM module loader, and AF_XDP zero‑copy networking. TLS 1.3 is enhanced with post‑quantum hybrid key exchange.

Core Coordination (K8s 1.28) – Summary

Deployed on Kubernetes 1.28, the Core uses Istio Service Mesh, Envoy API Gateway, and Apache Pulsar Functions for low‑latency stream processing. Authentication now supports OIDC and FIDO2 hardware keys.

Interface Layer (SPA < 25 KB) – Summary

The UI is a tiny vanilla‑JS SPA that leverages WebGPU for live charts and a responsive drawer that collapses into a hamburger menu on screens under 480 px. An integrated xterm.js terminal provides secure CLI access.

Security Model (Zero‑Trust 2.0) – Summary

Every inter‑service call uses short‑lived mTLS certificates, while OPA enforces policy‑as‑code RBAC. Data at rest is protected with XChaCha20‑Poly1305, and audit logs are immutable on IPFS.
