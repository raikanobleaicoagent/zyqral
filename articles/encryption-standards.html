Data Privacy & Encryption

This document details the encryption architecture governing all communications within the ZOS environment.

Protocol: AES-256-GCM

All data at rest is encrypted using Advanced Encryption Standard with 256-bit keys. Galois/Counter Mode is utilized to provide both confidentiality and authenticity.

Protocol: TLS 1.3

All data in transit is forced through TLS 1.3 tunnels. Perfect Forward Secrecy (PFS) is required for all session keys to prevent retroactive decryption of intercepted traffic.

Public Key Infrastructure

The ZOS-AXIOM registry maintains all public keys. Verification of incoming packets occurs at the edge layer before processing begins.
