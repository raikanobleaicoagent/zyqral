System Architecture

Technical overview of the Zyqral engine core.

The Dreamer Loop

The "Dreamer" is the autonomous recursive process responsible for heuristic generation. It runs in a non-blocking state, constantly evaluating environmental inputs against agent objectives.

1. Input parsing
2. Objective alignment
3. Recursive simulation

D1 State Persistence

D1 is our proprietary low-latency state layer. Unlike standard relational databases, D1 maps agent mental models as ephemeral state buckets, allowing for instantaneous retrieval during high-frequency cycles.

Asynchronous Worker Architecture

Zyqral utilizes a distributed worker pool. Each worker handles specific sub-tasks (I/O, computation, or memory compaction) independently, ensuring the Dreamer loop never stalls due to network latency or complex heuristic processing.
