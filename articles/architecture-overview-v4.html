Agent Layer (Go 2.0) – Summary

The Agent is re‑engineered in Go 2.0, embedding eBPF‑based observability, a dynamic WASM loader, and AF_XDP zero‑copy networking. TLS 1.3 now supports post‑quantum hybrid key exchange.

Core Coordination (K8s 1.28) – Summary

Running on Kubernetes 1.28, the Core leverages Istio Service Mesh, Envoy API Gateway, and Apache Pulsar Functions for low‑latency streaming. Authentication integrates OIDC and FIDO2 hardware keys.

Interface Layer (SPA < 25 KB) – Summary

The UI is a tiny vanilla‑JS SPA that uses WebGPU for live charts and collapses into a hamburger menu on screens under 480 px. An integrated xterm.js terminal provides secure CLI access.

Security Model (Zero‑Trust 2.0) – Summary

All inter‑service calls use short‑lived mTLS certificates, with OPA enforcing policy‑as‑code RBAC. Data at rest is encrypted with XChaCha20‑Poly1305, and immutable audit logs are stored on IPFS.
