Node Interaction Architecture

This document details the handshake and message propagation logic used by ZOS agents to maintain global consensus.

1. Initial Handshake

Nodes execute a standard diffie-hellman exchange followed by a capability advertisement broadcast.

> Encrypted handshake initialized.
> Key exchange: Curve25519.
> Signature algorithm: Ed25519.

2. Message Propagation

ZOS utilizes a gossip protocol optimized for low-bandwidth environments, ensuring message integrity via content-addressable hashing.

> Protocol: Anti-Entropy Gossip.
> Redundancy: 3-hop replication.
> Validation: Bloom filter lookups.
