Encryption Architecture

ZOS employs a layered security model designed to maintain data sovereignty in an untrusted network environment.

Layer 1: AES-256-GCM

All data at rest is encrypted using Advanced Encryption Standard with 256-bit keys in Galois/Counter Mode, ensuring both confidentiality and authenticity.

Layer 2: Zero-Knowledge Protocols

ZOS never stores plain-text credentials. We utilize salted hashing with Argon2id for all identity verification procedures.
